# 使用方法指南

> 作者正常的工作流程是这样的：  
> 拿到一道算法题后，新创建一个 AI 对话窗口，将题目内容原封不动完整喂给 AI，然后自己先独立敲一遍代码。  
> 如果不能 AC，则先将自己的思考过程抛给 AI，让 AI 进行初步复盘；但不要立刻看提示，而是自己先思考为什么 WA / TLE / RE。  
> 如果仍然想不出来，再参考 AI 的分析继续修改代码。  
> 如此往复，直到代码成功 AC。  
> 至此，代码部分就完成了。  
>
> 接下来，如何利用这个工具自动生成复盘文档呢？

---

# 📦 文件准备

创建一个新的文件夹，并将以下文件放置在同级目录：

```text
├── csp_word_generator_v9.py
├── input.txt
├── csp_input_template_v9.txt
```

随后，在该目录下安装 `requirements.txt` 内的依赖。

打开终端，在当前目录执行：

```bash
pip install python-docx matplotlib
pip install docx2pdf
```

---

# 🧠 AI 补全文档模板

在 AI 对话窗口中：

1. 将 `csp_input_template_v9.txt` 文件内容完整喂给 AI
2. 再将这道题相关的所有对话内容一起提供给 AI
3. 要求 AI：

> 以纯文本形式，结合所有对话内容，完整补全模板

---

# ✍️ 填写输入文件

将 AI 生成的文本内容：

- 全部复制
- 粘贴到 `input.txt` 文件内部

---

# 🚀 生成复盘文档

打开终端，在当前目录执行：

```bash
python csp_word_generator_v9.py input.txt
```

---

# 🎉 大功告成！

程序会自动生成：

- Word 文档（`.docx`）
- PDF 文档（`.pdf`）

完整算法复盘即可直接查看、打印或归档。
