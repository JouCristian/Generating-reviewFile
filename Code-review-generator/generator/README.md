# 使用方法指南

> 作者正常的工作流程是这样的：拿到一道算法题，新创建一个AI对话窗口，将内容原封不动完全喂给AI，然后自己先敲一遍代码，如果不能AC，则自己先将思考过程抛给AI，让AI进行初步复盘，然后自己不看AI提示自己先思考为什么不能AC，接着想不出来再看AI提示，然后再修改自己的代码、
> 如此往复，直到代码可以AC，这样整个代码的部分就结束了。接下来怎么利用这个工具生成复盘文档呢？

---
- 在AI窗口，将 *csp_input_template_v9.txt* 文件喂给它，然后让他以纯文本的形式，结合这道题的所有对话内容，补全模板内容
- 将生成的文本内容复制到 *input.txt* 文件内部
- 终端执行python文件 *python csp_word_generator_v9.py input.txt*

## 大功告成！已自动生成word和pdf文件
